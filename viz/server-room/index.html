01 机房平面布置图
俯视。机柜行背靠后墙(封闭热通道),正面朝冷通道,列头柜置行头(与 APC 机柜同尺寸);进线/UPS/电池贴入口端墙门左侧;FM200 控制盘进门右墙,吊顶布气体喷头 + 烟感;工作台顶 z=0 墙横放;冷凝器置屋顶。轴线、设备定位与检修间距见标注。

机柜行

6×42U + 3×InRow + 列头柜(行头,APC 同尺寸),行长 6500mm,C-R-R-R-C-R-R-R-C 间隔取热。

冷/热通道

热通道(贴后墙)封闭,冷通道≥1200mm 兼检修,送风经冷通道入柜。

强电(进门左墙)

进线柜 / UPS / 电池柜贴入口端墙门左侧,同高 2000,正面朝室内;列头柜已移入机柜行头。

气体灭火 FM200

控制盘 + 钢瓶贴进门右墙;吊顶布气体喷头 ×5 + 烟感 ×4,探测联动释放七氟丙烷。

出入口 / 屋顶 / 工作台

右端墙甲级隔音防火门 + 门禁;冷凝器上屋面(承重 + 防水);工作台顶 z=0 墙横放,双显示器支架。

03 配电系统图(单线)
三相市电进线 → 主开关 200A → 分两路:UPS(带电池)→ 强电列头柜 → 6× Panduit 机架 PDU;制冷/照明市电直供不上 UPS。

⚠ 供电前置条件与已知限制

  • 整机房含制冷 ≈88kW,需约 200A 三相;农村房须向供电申请增容 / 专变,否则无法满载。
  • UPS 为 单组(1 套),无 UPS 级 N+1 冗余 —— 建议选模块化 UPS 内置 N+1 以提升可用性。
  • 机架 PDU 当前单路供电;如需 2N,可增市电旁路第二路 PDU。
04 气流组织 / 散热示意
close-coupled 近端制冷:行级空调从封闭热通道取热风,经内部 DX 盘管换热后向冷通道送 22–24℃ 冷风,冷媒管上屋面经屋顶冷凝器排热。

风冷 DX · N+1

3 台 InRow RD(2 工作 + 1 备),实负载 60kW,单台标称 ~30kW;无水系统,免冷冻水管路。

热通道封闭

杜绝冷热气流掺混,提高回风温度与换热效率,PUE 更优。

屋顶冷凝器

屋面 3 台(钢支架架空),冷媒管穿屋面防水套管;需承重核算,冬季可配低温组件。
